# Modules Assignment - Python

This project demonstrates the use of **Python modules and packages**.

The assignment focuses on:
- Creating Python modules
- Importing modules
- Creating packages using `__init__.py`
- Using different import styles

No OOP, exceptions, or file handling are used.

---

## Project Structure

---

## Modules

### 1. math_utils.py
Contains simple mathematical functions:
- `add(a, b)` → returns sum
- `subtract(a, b)` → returns difference
- `square(n)` → returns square of number

---

### 2. string_utils.py
Contains string helper functions:
- `capitalize_words(text)` → capitalizes each word
- `reverse_string(text)` → reverses the string
- `word_count(text)` → counts number of words

---

## Package: shop_package

This package simulates a simple shop system.

### discount.py
- `apply_discount(price, percent)` → applies percentage discount
- `flat_discount(price)` → subtracts 50 from price

### billing.py
- `calculate_total(prices)` → returns total price
- `apply_tax(amount)` → adds 5% tax

### __init__.py
Allows direct importing of package functions.
