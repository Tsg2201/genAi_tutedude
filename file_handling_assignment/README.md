# Python File Handling Assignment

This project demonstrates **basic file handling operations in Python** such as writing, reading, appending, extracting data, and generating reports using simple built-in features.

## Topics Covered

- Opening files in different modes (`r`, `w`, `a`)
- Reading files using `read()`, `readline()`, and `readlines()`
- Writing and appending data to files
- Extracting data from text files
- Basic error-safe file handling
- Working with dictionaries and user input
- Generating simple reports

No advanced libraries such as **pandas or csv** are used.

---

# Project Tasks

## Task 1 – Write Sales Records
Creates a file `sales_data.txt` and writes a list of sales amounts into it.

## Task 2 – Read File in Different Ways
Reads the file using:
- `read()`
- `readline()`
- `readlines()`

## Task 3 – Append New Sales
Appends additional sales values to the existing file and prints the updated contents.

## Task 4 – Generate Sales Summary
Reads sales values from the file and calculates:
- Total Sales
- Highest Sale
- Lowest Sale
- Average Sale

## Task 5 – Product Info File
Takes product names and prices from the user and stores them in `products.txt`.
