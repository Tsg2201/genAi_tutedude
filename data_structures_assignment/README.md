# Python Data Structures Assignment

This repository contains solutions demonstrating the use of core Python data structures including **Lists, Tuples, Sets, and Dictionaries**.

## Project Structure

```
data_structures_assignment/
│
├── Task1.py
├── Task2.py
├── Task3.py
├── Task4.py
└── README.md
```

## Tasks Overview

### Task 1 – Product Collections (Lists & Tuples)

File: `Task1.py`

This script demonstrates:

* Creating a list of product names
* Creating a tuple representing a product `(product_name, price, category)`
* Printing the 2nd and last product in the list
* Appending new products to the list
* Converting a tuple to a list, modifying its price, and converting it back to a tuple

---

### Task 2 – Categories (Sets)

File: `Task2.py`

This script demonstrates:

* Creating a set of product categories
* Showing how sets remove duplicate values
* Adding a new category to the set
* Checking if a category exists in the set
* Finding the number of unique categories

---

### Task 3 – Product Pricing (Dictionaries)

File: `Task3.py`

This script demonstrates:

* Creating a dictionary mapping products to prices
* Adding new products with prices
* Updating the price of existing products
* Removing products safely
* Calculating the average price of products
* Identifying the product with the highest and lowest price

---

### Task 4 – Combined Operations

File: `Task4.py`

This script combines previous concepts to:

* Create a `catalog` list of tuples `(product_name, price, category)`
* Create a dictionary mapping categories to their products
* Identify the category containing the highest number of products
* Print all products in that category

---

## Requirements

* Python 3.x
